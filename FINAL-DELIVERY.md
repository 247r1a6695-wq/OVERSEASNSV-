## 🆘 TROUBLESHOOTING

### Website Won't Load
```
Solution: 
1. Clear browser cache (Ctrl+Shift+Del)
2. Try different browser
3. Check file paths are correct
4. Look for console errors (F12)
```

### Forms Not Working
```
Solution:
1. Check browser allows localStorage
2. Open DevTools (F12)
3. Check Console for errors
4. Verify form has all required fields
```

### Mobile Menu Not Working
```
Solution:
1. Check viewport meta tag present
2. Test on actual mobile device
3. Clear cache
4. Check JavaScript loaded correctly
```

### Admin Panel Won't Open
```
Solution:
1. Make sure you're logged in as admin
2. Use correct credentials (admin/admin123)
3. Check localStorage in DevTools
4. Try clearing localStorage and login again
```
