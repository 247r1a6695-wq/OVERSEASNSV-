## 🎯 SCORING TARGETS

### TAB 2: SEO Score
```
Target: 80%+

How to reach:
✓ Meta description: Exactly 150-160 chars
✓ Primary keyword: In title (title 30-60 chars)
✓ Content: Write 1500+ words
✓ Slug: Use lowercase, hyphens, no spaces
✓ Secondary keywords: Include 3-5
✓ Alt text: Add to all images

When you see GREEN (80%+): Ready for next tab!
```

### TAB 4: Content Score
```
Target: 80%+

How to reach:
✓ Word count: 1500+ words
✓ Headings: 5-10 H2/H3
✓ Images: 3+ with alt text
✓ Grammar: Use Grammarly - must be "Excellent"
✓ Readability: Use Hemingway - target 8th grade
✓ Depth: Comprehensive coverage of topic

When you see GREEN (80%+): Almost ready to publish!
```

### Overall Score
```
Target: 85%+

Formula: (SEO Score + Content Score) / 2

If both are 80%+ → Overall is 80%+ ✅
If both are 90%+ → Overall is 90%+ ✅✅✅ PREMIUM

Only publish when overall is 85%+
```
