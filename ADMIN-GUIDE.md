## 🔍 TAB 2: SEO OPTIMIZATION

### Primary SEO Fields:

#### 1. **SEO-Friendly Slug/URL** ⭐ CRITICAL
- Format: `my-blog-title-here`
- Only use: letters, numbers, hyphens
- This becomes your blog URL: `yoursite.com/blog/my-blog-title-here`
- **Must be unique** for each blog

#### 2. **Meta Description** (SERP Preview) ⭐ CRITICAL
- **Length:** 150-160 characters (shown in preview)
- **Purpose:** What people see in Google search results
- **Tips:** 
  - Include primary keyword naturally
  - Make it compelling - encourage clicks
  - Call-to-action optional but effective

#### 3. **Primary Keyword/Focus Keyword** ⭐ CRITICAL
- The main keyword you want to rank for
- Should appear in:
  - Title (H1)
  - First 100 words of content
  - Meta description
  - 1-2 times in body
- Example: "study abroad for Indian students"

#### 4. **Secondary Keywords** (Comma-separated)
- Related keywords to capture more searches
- Examples: "visa requirements, scholarship, universities"
- Don't overstuff - keep it natural

#### 5. **Focused Topic/Subtopic**
- Specific angle of your main topic
- Helps target long-tail keywords
- Example: "US Visa Interview Tips for Freshers"

### SEO Score (Real-time)
- **0-40%:** 🔴 Needs improvement
- **40-70%:** 🟠 Good
- **70-100%:** 🟢 Excellent

Automatically calculates as you fill in fields!

## ⚙️ TAB 3: ADVANCED SETTINGS

### Content Metrics (Auto-calculated):
- **Content Length (words)** - Total word count
- **Reading Time (minutes)** - At 200 words/min average

### Internal Linking
**What:** Links to other NSV Overseas blogs
**Why:** Improves SEO and keeps users on site
**How:** Add one link per line
```
/blogs/visa-tips
/blogs/universities
/colleges
```

### External Links
**What:** Links to authoritative external sources
**Why:** Builds credibility and improves rankings
**How:** Add one URL per line
```
https://visaaustralia.gov.au
https://www.qaa.ac.uk
https://studyportals.com
```

### Canonical URL (Optional)
**What:** URL if this content exists elsewhere
**Why:** Prevents duplicate content penalties
**Example:** `https://yoursite.com/articles/visa-guide`

### Open Graph Image (Social Media)
**What:** Image shown when shared on Facebook, Twitter, LinkedIn
**Why:** Better sharing and click-through rates
**Size:** 1200x630px recommended

### Schema Markup Type
**Purpose:** Helps Google understand content type
**Options:**
- **BlogPosting** - For blog articles
- **Article** - For news/editorial content
- **NewsArticle** - For time-sensitive news
- **WebPage** - General web pages

### Schema FAQs (Rich Snippets)
**Format:** JSON array
```json
[
  {
    "question": "How to apply for US visa?",
    "answer": "Visit the US Embassy website and follow the DS-160 form..."
  },
  {
    "question": "What documents are needed?",
    "answer": "Passport, I-20 form, financial documents..."
  }
]
```
**Result:** FAQ appears in Google search results!

- Write for your audience first, SEO second

- Publish consistently (weekly or bi-weekly)
- Update old blogs periodically
- Respond to seasonal events

- Include numbers in titles (e.g., "5 Tips...")
